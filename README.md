# 📊 Students Performance Analysis Dashboard

## 📌 Project Overview
This project analyzes student academic performance data to uncover trends, patterns, and key insights across different subjects and demographic categories. The dashboard was developed using Power BI to transform raw data into meaningful visual insights.

---

## 🎯 Objectives
- Analyze subject-wise performance (Math, Reading, Writing)
- Compare scores across gender and categories
- Evaluate the impact of test preparation
- Identify high and low-performing groups
- Generate actionable academic insights

---

## 📂 Dataset Description
The dataset includes the following key fields:

- Math Score
- Reading Score
- Writing Score
- Gender
- Parental Level of Education
- Lunch Type
- Test Preparation Course
- Other demographic attributes

The data was cleaned and transformed before visualization.

---

## 🛠 Tools & Technologies Used
- Power BI
- DAX (Data Analysis Expressions)
- Excel / CSV
- GitHub

---

## 📊 Key Metrics
- Average Math Score
- Average Reading Score
- Average Writing Score
- Total Students
- Performance by Gender
- Performance by Test Preparation
- Category-wise Comparison

---

## 📈 Key Insights
- Identified subject with highest and lowest average score
- Analyzed impact of test preparation on performance
- Compared academic performance across demographic groups
- Observed overall academic trends

---

## 📊 Dashboard Features
- KPI Cards
- Bar & Column Charts
- Pie / Donut Charts
- Slicers (Interactive Filters)
- Clean and Professional Layout

---

## 🚀 Outcome
This dashboard provides clear insights into student performance and supports data-driven educational decision-making.
